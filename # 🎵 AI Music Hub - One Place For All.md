# 🎵 AI Music Hub - One Place For All

An intelligent music recommendation system powered by AI, featuring multi-language support and weather-based recommendations.

## ✨ Features

- 🎼 **Smart Music Recommendations** - AI-powered song suggestions based on your preferences
- 🌤️ **Weather-Based Playlists** - Music that matches your local weather and mood
- ✍️ **Multi-Language Lyrics Generator** - Generate lyrics in 10+ languages (English, Hindi, Kannada, Tamil, Telugu, Malayalam, Marathi, Bengali, Punjabi, Gujarati)
- 🎹 **AI Song Creator** - Generate detailed song descriptions for AI music platforms
- 🎵 **217+ Songs Database** - Diverse collection including Kannada, Hindi, Tamil, Telugu, and international songs

## 🚀 Quick Start

### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)

### Installation

1. Clone the repository:
```bash
git clone <your-repo-url>
cd AI-Music-Hub-New
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

Or simply double-click `start.bat` (Windows)

4. Open your browser:
```
http://127.0.0.1:5001
```
